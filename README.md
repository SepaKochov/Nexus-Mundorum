# Nexus Mundorum
Краткое описание вселенной Nexus Mundorum и её назначения.
Перечень контента: существа, боги, артефакты, локации и другие записи.
Онлайн: https://sepakochov.github.io/Nexus-Mundorum/
Запуск локально: открыть index.html в браузере.
Админка: файл admin.html в корне.
Данные: папка data/.
Фронтенд-код: папка js/.
Стиль работы: изменения через git commit/push.
Статический сайт без сборки (HTML/CSS/JS).
